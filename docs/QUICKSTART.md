# AI-Agent Allowance Controls Demo

A local-first TypeScript demo for Solana Native Subscriptions & Allowances.

## One-command check

```bash
npm install
npm run check
```

## Core files

- `README.md` — main packet
- `docs/architecture.md` — design and devnet mapping
- `docs/references.md` — source alignment notes
- `docs/submission-draft.md` — local draft, not submitted
- `src/mock-demo.ts` — runnable mock demo
- `tests/allowance-policy.test.ts` — automated policy tests

## Current boundary

Mock/local only. No keys, no wallets, no RPC calls, no transactions, no submission.
